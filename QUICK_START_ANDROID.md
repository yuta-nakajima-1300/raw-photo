# 🚀 Android実機テスト - クイックスタート

## ⚡ 最速実行手順（5分で完了）

### 1. Android端末の準備（2分）
```
📱 Android端末で:
1. 設定 > 端末情報 > ビルド番号を7回タップ
2. 設定 > 開発者向けオプション > USBデバッグをON
3. USBケーブルでPCに接続
```

### 2. PC側の確認（1分）
```bash
# Android端末が認識されているか確認
adb devices

# 期待される出力:
# ABC123DEF456    device
```

### 3. アプリ実行（2分）
```bash
cd flutter_raw_editor/raw_photo_editor

# 依存関係インストール（初回のみ）
flutter pub get

# 実機で実行
flutter run

# または高速リリースモード
flutter run --release
```
